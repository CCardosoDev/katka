# Katka

Frontend for Katka release orchestrator.

## Dependencies
* Node.js 8
* npm
* React 16.3.2
* Semantic UI
* Jest
* eslint
* Babel
* Webpack

## Run tests locally

```shell
    $ make install_dependencies
    $ make test_local
```

## Run tests on docker container

```shell
    $ make test
```

## Run locally

```shell
    $ make run_local
```

## Build application

```shell
    $ make build
```
